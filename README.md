# SenseReact

テックシーカーハッカソン2026 出展作品

## これは何？

SenseReact（センスリアクト）は「**かざして感じ、かざして動かす**」前方感応型の身体拡張デバイス。
人が感じ取れない前方の情報をセンサーで「察知」し、振動で身体に伝え、手のポーズで外部マシンを「動かす」までを一連で実現する。

**動作フロー：**

1. **察知** … 超音波センサー・HDRカメラで、手をかざした先を捉える
2. **フィードバック** … 振動モーターで距離・気配を手首に伝える
3. **ポーズ検知** … 曲げセンサー・加速度センサーで手の構えを読み取る
4. **出力** … サーボ／ラジコンなど外部マシンが動く

別系統として、地磁気センサーで方角を振動で感じる「方位の第六感」も搭載予定。

## 進捗（2026-06-18）

**✅ 終わってる**
- カメラ：撮影（VGA）・動体検知
- 入力センサー：曲げ・超音波・9軸IMU
- 出力：連続回転サーボ・振動モーター
- 超音波 → 振動（近いほど強く）＝ MVP前半
- 無線：ESP-NOW疎通・車体サーボを無線で制御

**⬜ 終わってない**
- MVP統合（曲げ → サーボを繋いで一連に）
- SPRESENSE ↔ ESP32 ブリッジ（手で車を操縦）
- ステアリング（IMU）・方位の第六感（磁気 ＋ 内蔵GPS）
- IMUでカメラ差分の精度向上（自分の動きを除外）

## キャリブレーション値（忘れないこと）

### 曲げセンサー（MB090-N-221-A02 / A3ピン）

| 項目 | 値 |
|---|---|
| フラット時ADC（0%） | 840 以上 |
| 100%閾値ADC | 580 以下 |
| BEND_MAX_DIFF | 260 |

### 連続回転サーボ（D06ピン）

| 項目 | μs |
|---|---|
| 停止 | 1500 |
| MIN速度（5%） | 1510 |
| MAX速度（100%） | 2500 |

## ピン配置

| 部品 | ピン | ボード |
|---|---|---|
| 曲げセンサー | A3 | メインボード |
| サーボ Signal | D06 | 拡張ボード |
| サーボ VCC | VOUT（5V） | 拡張ボード |
| サーボ GND | GND | 拡張ボード |

## スケッチ一覧

| フォルダ | 内容 |
|---|---|
| HDR_Camera_Capture | カメラ撮影確認（VGA動作済み） |
| Camera_MotionDetect | カメラ動体検知（フレーム差分・察知の基礎） |
| Bend_Sensor_Test | 曲げセンサー単体テスト |
| Servo_Test | サーボ単体テスト（スイープ） |
| Bend_Servo_Control | 曲げ→サーボ速度連動（メイン） |
| Ultrasonic_Test | 超音波距離計測（US-015） |
| Vibration_Test | 振動モーター制御テスト（PWM強さ） |
| Ultrasonic_Vibration | 超音波×振動 統合（近いほど強く・MVP前半） |
| ESP32_Blink | ESP32動作確認（Lチカ） |
| ESPNow_Receiver | ESP-NOW受信テスト（ラジコン無線リンク） |
| ESPNow_Sender | ESP-NOW送信テスト（ラジコン無線リンク） |
| Car_Receiver | 車体ESP32：ESP-NOW受信→連続回転サーボ駆動（差動・停止1500/全開2500） |
| Drive_Test_Sender | 車体テスト送信：前進/旋回/停止を1.5秒ごとに送る |
| MPU9250_Motion | 9軸IMU：加速度・ジャイロ各軸＋腕の動き量（I2C直読み） |
