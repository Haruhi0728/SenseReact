# SenseReact

テックシーカーハッカソン2026 出展作品

## キャリブレーション値（忘れないこと）

### 曲げセンサー（MB090-N-221-A02 / A3ピン）

| 項目 | 値 |
|---|---|
| フラット時ADC（0%） | 840 以上 |
| 100%閾値ADC | 580 以下 |
| BEND_MAX_DIFF | 260 |

### 連続回転サーボ（D06ピン）

| 項目 | μs |
|---|---|
| 停止 | 1500 |
| MIN速度（5%） | 1510 |
| MAX速度（100%） | 2500 |

## ピン配置

| 部品 | ピン | ボード |
|---|---|---|
| 曲げセンサー | A3 | メインボード |
| サーボ Signal | D06 | 拡張ボード |
| サーボ VCC | VOUT（5V） | 拡張ボード |
| サーボ GND | GND | 拡張ボード |

## スケッチ一覧

| フォルダ | 内容 |
|---|---|
| HDR_Camera_Capture | カメラ撮影確認（動作済み） |
| Bend_Sensor_Test | 曲げセンサー単体テスト |
| Servo_Test | サーボ単体テスト（スイープ） |
| Bend_Servo_Control | 曲げ→サーボ速度連動（メイン） |

## ビルド環境

- arduino-cli v1.5.1
- SPRESENSE:spresense v3.4.7
- ボード: `SPRESENSE:spresense:spresense`
- ポート: COM4（環境によって変わる）
